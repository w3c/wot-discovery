Web of Things (WoT) Discovery Ontology

Work in Progress

An HTML rendering of the Web of Things (WoT) Discovery ontology is under development. In the meantime, please refer to the Turtle (ttl) file which can be fetched from this same URL, https://www.w3.org/2022/wot/discovery-ontology, but using the text/turtle Content-Type.

You can also find this file on github at https://github.com/w3c/wot-resources/blob/main/discovery/v1/ontology/discovery-ontology.ttl.
